Descrição do Projeto

Implementação em Python dos Padrões de Projeto Método Fabrica e Protótipo, adicionalmente foi implementado os padrões Fabrica Abstrata e Builder. Todos os padrões foram utilizados em projeto que se baseia no exemplo do livro que ilustra uma fabrica de veículos e funcionam de forma combinada.
